1. selenium

셀레니움은 웹 브라우저를 코드 기반으로 실제 조작하는 방식으로 이용할 수 있는 python의 프레임워크입니다.

다양한 브라우저(Chrome, Firefox 등), 다양한 언어(Java, Python, PHP 등)를 지원하는데 이 글에서는 크롬 기반으로 진행해보겠습니다.


2. 기본 환경을 위한 다운로드

2-1) 가상환경 설정

여러 버전의 Python을 실행하기 위해 가상환경 설정을 했는데 버전 하나로만 실행하시는 분들은 바로 아래 단계로 넘어가면 됩니다.


2-2 selenium 설치

가상 환경에서 pip install selenium 을 입력 후 실행하면 됩니다.


2-3) 크롬 드라이버 설치

구글에서 크롬 드라이버를 검색 후 홈페이지)에 들어갑니다. 현재 사용하고 있는 크롬 브라우저의 버전을 확인 후 다운로드하면 됩니다. 현재 버전을 확인하려면 크롬 설정에 들어가서 Chrome 정보를 클릭해서 확인할 수 있습니다.

버전 확인 후 다시 크롬 드라이버 홈페이지에서 자신의 버전과 맞는 링크를 클릭 후 다운로드 하시면 됩니다. 윈도우에서 실행할 것이기 때문에 저는 윈도우 버전을 받았습니다.

zip 파일을 다운로드했으면 압축을 풀고 아까 만들었던 selenium 폴더로 exe 파일을 옮기면 모든 준비가 완료됩니다.


3. 이미지 크롤링 시도

셀레니움을 실행하기 위한 환경이 완성이 되었습니다. 이제 실행해보도록 합시다.

3-1 selenium 테스트

큰 이상이 없으면 크롬 브라우저가 실행되어 구글 메인화면이 나온 뒤 브라우저가 자동으로 종료되었을 것입니다.

browser.get 안에 기존 주소 대신에 구글 이미지 주소를 입력해봅시다.
