Plan

블로그의 계획을 간략하게 요약한 페이지 입니다. 현재는 복습 목적으로 블로그 작성중이며 복습이 끝나고 나면 최근에 배운 내용을 업로드하면서 기존에 작성한 글들의 가독성을 높여 데이터 분석에 입문하는 사람도 쉽게 읽을 수 있도록 편집하는 것이 목표입니다.

데스크탑으로 볼 때는 원하는 대로 코드가 입력되고 있는데 휴대폰으로 블로그에 방문하면 SQL이나 Python 코드를 적기 위한 highlights.js 서식이 제대로 적용되지 않는 모습을 발견했습니다. 해결 방법을 찾아보겠습니다.
